# 🎨 Site Tataah Arts

Projeto desenvolvido com foco na criação de um site visualmente atrativo para apresentação de artes digitais, com identidade voltada para um público criativo e sensível ao design.

---

## 🚀 Sobre o projeto

O **Tataah Arts** é um site pensado para divulgação de trabalhos artísticos, com foco em estética, leveza e experiência do usuário.
A proposta foi criar uma interface que valorize as imagens e transmita personalidade através das cores, tipografia e interações.

---

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript
* Organização de assets (imagens, ícones e mídias)

---

## 🎯 Objetivos do projeto

* Criar uma interface visual agradável e coerente com a proposta artística
* Trabalhar conceitos de **design aplicado ao front-end**
* Desenvolver estrutura semântica em HTML
* Aplicar estilização com foco em identidade visual
* Implementar interações simples com JavaScript

---

## 💡 Funcionalidades

* Exibição de artes e elementos visuais
* Layout responsivo (em construção/melhorias futuras)
* Estrutura organizada para expansão do projeto
* Uso de ícones e imagens personalizadas

---

## 🎨 Identidade visual

O projeto utiliza uma paleta de cores suaves e harmônicas, buscando transmitir:

* Delicadeza
* Criatividade
* Expressividade artística

---

## 📂 Estrutura do projeto

```bash
assets/        # imagens e mídias
icon/          # ícones do site
mascote/       # elementos visuais personalizados
index.html     # estrutura principal
styles.css     # estilização
script.js      # interações
```

---

## 📈 Melhorias futuras

* Responsividade completa para dispositivos móveis
* Animações e interações mais avançadas
* Integração com formulário de contato
* Otimização de performance e imagens
